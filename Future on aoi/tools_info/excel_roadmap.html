Essential Excel Functions & Features

Logical Functions

Make decisions in your spreadsheet.

  • IF: Performs a logical test.
  • AND / OR: Combine multiple conditions.
  • IFERROR: Catches and handles errors gracefully.

Lookup & Reference

Find and retrieve data from other locations.

  • VLOOKUP / HLOOKUP: The classic way to find data in a table.
  • INDEX & MATCH: A more powerful and flexible combination.
  • XLOOKUP: The modern, superior successor to VLOOKUP.

Text Functions

Manipulate and clean text data.

  • CONCAT / &: Join text strings together.
  • LEFT / RIGHT / MID: Extract parts of a text string.
  • TRIM: Remove extra spaces from text.

Data Analysis Power Tools

Conditional Formatting

Automatically apply formatting (colors, icons) to cells based on their values to highlight trends and outliers.

PivotTables

Summarize, group, count, and analyze vast amounts of data with a simple drag-and-drop interface. The single most powerful analysis tool in Excel.

Data Visualization

Create impactful charts (Bar, Line, Pie, Scatter) to tell a story with your data and make it easy for others to understand.

Level Up: Advanced & Automation Skills

For those who want to become true Excel power users, these tools separate the proficient from the experts. They are designed to handle large-scale data and automate repetitive work.

Power Query

An incredibly powerful data connection and transformation tool. Use it to import, clean, and reshape data from almost any source before it even enters your spreadsheet. It's the foundation of modern data analysis in Excel.

Macros & VBA

Automate any repetitive task by recording your actions or writing code in Visual Basic for Applications (VBA). If you do something more than three times, it's a candidate for a macro to save you hours of work.
