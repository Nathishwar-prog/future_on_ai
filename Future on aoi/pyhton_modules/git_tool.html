Visual Git Command Guide

A quick reference for the most essential Git & GitHub commands.

A Developer's Daily Toolkit

git clone

Copy a remote repository.

git clone [url]

git status

Check the state of your repo.

git status

git add

Stage changes for commit.

git add [file] | git add .

git commit

Save staged changes locally.

git commit -m "Your message"

git pull

Fetch & merge remote changes.

git pull origin main

git push

Share local commits remotely.

git push origin [branch-name]

Branching & Integration

git branch

Create or list branches.

git branch [new-branch-name]

git checkout

Switch to a different branch.

git checkout [branch-name]

git merge

Combine histories of branches.

git merge [branch-to-merge]

Problem-Solving Arsenal

git rebase

Re-apply commits on a new base.

git rebase [base-branch]
Private Branches Only

git reset

Undo commits, erasing history.

git reset [--soft|mixed|hard]
Destructive

git revert

Undo a commit with a new one.

git revert [commit-hash]
Safe for Public Branches

git stash

Temporarily save changes.

git stash | git stash pop
Local Workspace

git cherry-pick

Apply a specific commit.

git cherry-pick [commit-hash]
For Hotfixes

git reflog

Your ultimate safety net.

git reflog
Local Recovery
