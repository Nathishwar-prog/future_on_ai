OS Module Core Functions

  • os.chdir() - Change working directory
  • os.getcwd() - Get current working directory
  • os.listdir() - List directory contents
  • os.mkdir()/makedirs() - Create directory(ies)
  • os.remove()/unlink() - Delete file
  • os.rename() - Rename/move file/directory
  • os.rmdir()/removedirs() - Remove directory(ies)
  • os.path.exists() - Check if path exists
  • os.path.join() - Join path components
  • os.path.basename()/dirname() - Get path components
  • os.path.isdir()/isfile() - Check path type
  • os.path.getsize() - Get file size
  • os.stat() - Get file status
  • os.environ - Access environment variables
  • os.system() - Execute shell command
  • os.walk() - Directory tree generator
  • os.access() - Check path permissions
  • os.urandom() - Generate random bytes

SYS Module Core Functions

  • sys.argv - Command line arguments
  • sys.exit() - Exit interpreter
  • sys.path - Module search path
  • sys.modules - Loaded modules
  • sys.stdin/stdout/stderr - Standard streams
  • sys.version - Python version info
  • sys.platform - OS platform identifier
  • sys.getsizeof() - Object memory usage
  • sys.exc_info() - Exception information
  • sys.settrace() - Set trace function

Sys Module: Script & Interpreter

sys.argv

The gateway to all command-line arguments passed to your script.

$ python script.py arg1

↓

['script.py', 'arg1']

sys.path

Crucial for understanding and manipulating where Python looks for modules.

→
['/app', '/lib', ...]

sys.exit()

The canonical way to terminate a script, optionally with an error code.

→

sys.platform

Allows you to write OS-specific logic (e.g., for 'win32', 'linux', or 'darwin').

Direct access to the script's input/output streams, vital for shell pipelines.

Important Functions (Supporting Roles)

Frequently used functions that support critical operations and handle common tasks.

os.path.isdir() / isfile()

Frequent and necessary checks to determine if a path is a directory or a file.

if os.path.isfile(p): ...
→ isfile? →
True
→ isdir? →
True

os.rename()

A common and essential file operation for renaming or moving files and directories.

os.rename('a.txt', 'b.txt')
'old.txt'
→
'new.txt'

os.remove()

The most basic and direct way to delete a file from the filesystem.

os.remove('stale.log')

os.walk()

Powerful generator for recursively traversing a directory tree from top to bottom.

for r, d, f in os.walk('.'): ...

↓

yields (root, dirs, files)

os.environ

Dictionary-like access to system environment variables for configuration.

key = os.environ.get('API_KEY')
→
{'API_KEY': '...'}

sys.stdin / stdout / stderr

File objects for script I/O, essential for creating command-line tools.

sys.stdout.write('Hello')

Input → [stdin] → Script

Script → [stdout] → Output

Script → [stderr] → Error

sys.version

Check the running Python version for compatibility logic.

if sys.version_info.major < 3: ...
→
'3.10.4 (...)'

os.stat()

Retrieve detailed file metadata like size, permissions, and modification times.

info = os.stat('file.txt')
→
st_size, st_mtime...

Utility Functions (Specialized Tools)

Valuable for edge cases, convenience, or specific applications.

os.path.getsize()

A nice-to-have for getting the size of a file in bytes.

size = os.path.getsize('file.txt')
→
1024 (bytes)

os.urandom()

Specialized use for generating cryptographically strong random bytes.

key = os.urandom(16)
→
b'\x12\xab...'

os.access()

Used for checking user's access permissions for a path (Read, Write, Execute).

can_write = os.access('.', os.W_OK)
→ W_OK? →
True

sys.getsizeof()

Useful for memory optimization by checking the size of a Python object in bytes.

mem = sys.getsizeof([1, 2, 3])
[1,2,3]
→
80 (bytes)

sys.exc_info()

Provides detailed information about the current exception for advanced debugging.

except:
    type, val, tb = sys.exc_info()
→
(type, val, tb)

os.system()

Executes a shell command. Often better alternatives exist (like `subprocess`).

os.system('echo "Hello"')
→
Shell Output

os.path.splitext()

Helpful edge case for splitting a path into its root and extension.

os.path.splitext('img.png')
'image.png'
→
('image', '.png')

sys.settrace()

Advanced debugging hook to trace program execution, used by debuggers.

sys.settrace(my_trace_func)
→
Debugger Hook
