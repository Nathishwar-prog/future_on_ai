datetime Module Functions

datetime.now()

Returns the current local datetime (timezone-aware if possible).

from datetime import datetime
# Returns: 2025-07-08 19:53:00
now = datetime.now()

datetime.strftime()

Formats a datetime object into a string.

now.strftime("%a, %H:%M")

datetime.strptime()

Parses a string into a datetime object.

datetime.strptime("2025-07-08", "%Y-%m-%d")

timedelta()

Represents a duration for time arithmetic.

from datetime import timedelta
deadline = now + timedelta(days=10)

date()

Creates a date object (year, month, day).

time()

Creates a time object (hour, minute, second).

datetime()

Creates a full datetime object.

datetime.combine()

Combines a date and a time object.

datetime.fromisoformat()

Parses an ISO 8601 string.

datetime.fromtimestamp()

Converts a POSIX timestamp.

datetime.timestamp()

Converts to a POSIX timestamp.

datetime.today()

Current local datetime (naive).

time Module Functions

import time
start_time = time.time()

Converts timestamp to UTC struct_time.

time.localtime()

time.mktime()

Converts struct_time to a timestamp.

time.ctime()

Converts a timestamp to a string.

time.perf_counter()

High-resolution performance counter.

time.monotonic()

A clock that cannot go backward.

Critical Datetime & Time Functions (High Impact)

The essential functions for handling time-related operations in Python.

datetime.now() / .today()

Essential for getting the current time, used in logging, timestamps, and scheduling.

from datetime import datetime
now = datetime.now()

datetime.strftime()

Critical for formatting dates into human-readable strings (e.g., logs, reports).

now.strftime("%b %d, %Y")

datetime.strptime()

Key for parsing strings into datetime objects (e.g., user input, file parsing).

datetime.strptime("2025-07-08", "%Y-%m-%d")

timedelta()

Fundamental for time arithmetic (e.g., calculating deadlines, time differences).

from datetime import timedelta
deadline = now + timedelta(days=10)

time.time()

Core for benchmarking, timing operations, and simple timestamps (Unix epoch time).

import time
start_time = time.time()

time.sleep()

Essential for delays, rate limiting, and synchronization.

import time
time.sleep(5) # Pauses for 5 seconds

Important Functions (Supporting Role)

Frequently used functions that build upon the critical set.

datetime.fromtimestamp()

Create a datetime from a POSIX timestamp.

dt_obj = datetime.fromtimestamp(1751998320)

datetime.combine()

Combine a separate date and time into one object.

d = date(2025, 7, 8)
t = time(19, 52)
dt = datetime.combine(d, t)

datetime.timestamp()

Convert a datetime object to a POSIX timestamp.

ts = datetime.now().timestamp()

time.localtime() / .gmtime()

Convert timestamp to local or UTC time struct.

local = time.localtime() # From current time
utc = time.gmtime(1751998320)

Utility Functions (Edge Cases & Convenience)

Useful for specific or advanced scenarios.

time.perf_counter()

High-resolution timer for performance measurement.

start = time.perf_counter()
# ... some code ...
end = time.perf_counter()

time.process_time()

Returns CPU time of the current process.

cpu_time = time.process_time()

time.ctime()

Convert a timestamp to a pre-formatted string.

time_str = time.ctime(1751998320)

datetime.utcnow()

Naive UTC time; deprecated in favor of timezone-aware now().

from datetime import timezone
now_utc = datetime.now(timezone.utc)
